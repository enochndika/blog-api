This is an [Express.js](https://expressjs.com/) API
## Getting Started

First install dependencies
```bash
npm install
# or
yarn install
```
Then, in ```.env file``` add your following keys:

- database name 
- database username
- database password
- cloudinary api key
- cloudinary api secret
- cloudinary cloud name
- jwt secret


Then, run the development server:
```bash
npm run dev
# or
yarn dev
```
